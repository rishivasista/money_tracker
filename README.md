This is a MERN Stack app and it is live on https://money-tracker-rishi.netlify.app/

To use this app enter the data as following:
If you want to deduct some amount say 20 for chocolates
then write it as -20 ForChocolates.
Note: There should be only one space in purpose.

and if you want to credit amount do the following:
+100 Credit

Note:The signs(+ve and -ve) are important otherwise the app will not work.
